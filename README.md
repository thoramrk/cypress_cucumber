# cypress-cucumber-example
Initial example of using Cypress with Cucumber.

```
> Clone or download the framework
> navigate to project folder directory
> npm install >> Install Dependencies

# Run example tests
> npm run testall >> Execute all the feature files in headless mode
> npm run test-search-feature >> Executing a particular test using chrome
> npm run test-all-feature >> Execute all the feature files using browser chrome
> npm run test-tag-smoke >> Execute tests based on tags
> npm run report >> generate cucumber html report
